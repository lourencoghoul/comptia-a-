# Hertz (Hz)

## Definition
Hertz (Hz) is a unit of frequency that measures how many cycles occur in one second.

1 Hz = 1 cycle per second

## How it works
Frequency represents how often something repeats over time.

In computing, Hertz is used to measure how fast a processor operates.

## CPU Example
A processor with 3.0 GHz (Gigahertz) means:
- 3 billion cycles per second

## Common Units
- Hz (Hertz) = 1 Hz
- kHz (Kilohertz) = 1,000 Hz
- MHz (Megahertz) = 1,000,000 Hz
- GHz (Gigahertz) = 1,000,000,000 Hz

## Where Hertz is used
- CPU clock speed
- RAM frequency
- Monitor refresh rate (Hz)

## Key Point
Higher Hz usually means faster operation, but performance also depends on other factors.
