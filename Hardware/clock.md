# CPU Clock

## Definition
The clock is an electrical signal that controls the timing of the CPU.

## How it works
The clock generates regular pulses that synchronize all CPU operations.

Each pulse is called a clock cycle.

## Relationship with Hertz
The clock speed is measured in Hertz (Hz), which represents how many cycles occur per second.

Example:
- 1 Hz = 1 cycle per second
- 3 GHz = 3 billion cycles per second

## Why it matters
The clock determines how fast the CPU can execute instructions.

Higher clock speed generally means faster processing, but performance also depends on other factors such as cores and architecture.

## Key Point
The clock controls timing, while Hertz measures its frequency.
