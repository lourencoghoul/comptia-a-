#  Basic IT Toolkit

##  Common Tools

* **Toolkit** → Set of essential tools used for computer maintenance and repair
* **Thumb Drive (USB Flash Drive)** → Portable storage device used to transfer files and install operating systems
* **Voltage Tester** → Device used to check the presence of electrical voltage
* **Nut Driver** → Tool used to tighten or loosen hex nuts and standoffs inside a computer case
* **Voltmeter** → Instrument used to measure electrical voltage in circuits

---

##  Usage

* Diagnose hardware issues
* Install and repair components
* Test electrical safety
* Perform maintenance tasks

---

##  Note

These tools are essential for IT technicians and are commonly required knowledge for the CompTIA A+ certification.
