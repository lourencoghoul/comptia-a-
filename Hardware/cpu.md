#  CPU (Central Processing Unit)

##  Overview

The CPU (Central Processing Unit) is the main component of a computer responsible for processing instructions and performing calculations. It operates using binary language (0s and 1s), which is the fundamental way computers process data.

---

##  Functions

* Executes program instructions
* Performs calculations and logical operations
* Manages system processes
* Communicates with memory and hardware

---

##  Main Components

* **ALU (Arithmetic Logic Unit)** → Performs mathematical and logical operations

The ALU is the part of the CPU that performs calculations and logical operations.
It works with binary data (0s and 1s), executes arithmetic and logic tasks, and returns the result to the CPU.

* **CU (Control Unit)** → Directs operations within the CPU

The CU (Control Unit) is responsible for controlling and coordinating all CPU operations. It interprets instructions, directs data between components (such as the ALU, memory, and registers), and ensures everything is executed in the correct order.

* **Registers** → Small, fast storage locations inside the CPU

Registers are small, high-speed storage locations inside the CPU used to temporarily hold data and instructions during processing. They are the fastest type of memory and are essential for efficient CPU operation.

In x86 architecture, common general-purpose registers include:

AX (Accumulator) → Handles arithmetic operations and stores results
BX (Base) → Used for memory addressing
CX (Count) → Controls loops and counting operations
DX (Data) → Used in data transfer and input/output operations

Registers work closely with the ALU and Control Unit, allowing the CPU to quickly access and process data without relying on slower memory like RAM.
